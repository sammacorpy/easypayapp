# easypayapp

## Setup Backend
1.  clone this repository
    ```
    git clone https://github.com/sammacorpy/easypayapp.git
    ```

2.  change working directory to easy_pay_app_backend
    ```
    cd easy_pay_app_backend
    ```

3.  Install all dependencies
    ```
    npm install
    ```
